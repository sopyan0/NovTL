
PANDUAN MENGAKTIFKAN BUILD OTOMATIS
===================================

Karena folder berawalan titik (.github) sulit dibuat di komputer lokal, folder ini dinamakan "github_setup".

LANGKAH WAJIB SETELAH UPLOAD KE GITHUB:
1. Buka repository GitHub kamu di browser.
2. Cari folder bernama "github_setup".
3. Klik titik tiga (...) atau opsi rename.
4. Ubah nama folder "github_setup" menjadi ".github" (pakai titik di depan).
5. GitHub akan otomatis mengenali folder ini sebagai workflow dan mulai memproses aplikasi kamu.

Cek tab "Actions" di GitHub untuk melihat proses pembuatan aplikasi Android & Windows.
